### Listing Processes and Information


```bash
# examine processes that are associated with your session
ps
    PID TTY          TIME CMD
  18673 pts/0    00:00:00 bash
  18913 pts/0    00:00:00 ps

# examine specific process
ps -p

# more info 
ps -f
```
